# Генератор надёжных ключей (multi-pass-generator)

Инструмент для массовой генерации криптографически стойких паролей с гибкой настройкой параметров для каждого блока. Позволяет создавать сотни уникальных комбинаций за один запуск с автоматическим экспортом в структурированные папки.

## Возможности

- Генерация нескольких независимых блоков паролей за один запуск
- Поддержка латиницы, кириллицы и армянского алфавита (строчные и заглавные)
- Криптографически безопасный генератор на базе модуля `secrets`
- Автоматическое создание структурированных папок с двумя вариантами файлов (с нумерацией и без)
- Тёмная и светлая темы оформления интерфейса
- Двуязычный интерфейс (Русский/Английский)
- Полностью автономная работа без передачи данных по сети
- Возможность сборки в автономный EXE-файл без установки Python

## Требования

- Python 3.6 или выше
- Библиотека Pillow (для работы с иконкой приложения)

## Установка из исходного кода

1. Клонируйте репозиторий:
   `git clone <repository-url>`
2. Перейдите в папку проекта:
   `cd bulk-password-generator`
3. Создайте и активируйте виртуальное окружение:
   `python -m venv venv`
   `venv\Scripts\activate` (Windows) 
4. Установите зависимости:
   `pip install -r requirements.txt`
5. Запустите приложение:
   `python main.py`

## Использование

1. Настройте параметры блока: укажите количество паролей, длину и выберите нужные наборы символов.
2. При необходимости добавьте дополнительные блоки кнопкой "Добавить блок".
3. Нажмите "Сгенерировать" и выберите целевую папку.
4. Результаты будут автоматически сохранены в структуре `generated_keys/`.

## Формат вывода

Для каждого блока создаётся отдельная директория, содержащая:
- `keys.txt` — список паролей без нумерации
- `keys_numbered.txt` — список паролей с порядковой нумерацией
- `README.txt` — журнал настроек и время генерации

## Сборка автономного файла (Windows)

Для создания портативной версии используйте встроенныq скрипт: `build.py` 

## Безопасность

- Генерация основана на криптографически стойком модуле `secrets`
- Приложение не требует подключения к интернету
- Данные не передаются на внешние серверы и не логируются
